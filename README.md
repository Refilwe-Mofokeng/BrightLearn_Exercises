# SQL Fundamentals Exercises

## Overview
This repository contains a series of SQL exercises designed to build and strengthen foundational data analysis skills using structured queries.

---

## Structure of Exercises

There were four exercises, each targeting a fundamental SQL skill set:

### 1. SELECT & Filtering Statements
Focused on retrieving and filtering data from tables using:
- `SELECT`
- `DISTINCT`
- `ORDER BY`
- `LIMIT`
- `WHERE`
- Logical operators (`AND`, `OR`, `NOT`,)
- `IN`, `BETWEEN`,
Combining the different SQL functions

Objective:  
Learn how to extract relevant subsets of data based on specific conditions.

---

### 2. Aggregate Functions & Grouping
Explored how to summarise data using:
- `SUM`
- `COUNT`
- `AVG`
- `MIN`
- `MAX`
- `ORDER BY`
- `GROUP BY`
- `HAVING`

Objective:  
Understand how to aggregate data to generate insights at a grouped level (e.g., totals, averages, counts).

---

### 3. CASE Statements
Introduced conditional logic within SQL queries:
- `CASE WHEN` statements
- Creating derived columns
- Categorising data dynamically

Objective:  
Enable more advanced analysis by transforming raw data into meaningful segments (e.g., categories, buckets).

---

### 4. Joins, Unions, Filtering & Aggregation
Combined multiple SQL concepts to solve more complex problems:
- `INNER JOIN`
- `LEFT JOIN`
- `RIGHT JOIN`
- `UNION`
- Filtering with `WHERE`
- Logical operators (`AND`, `OR`, `NOT`, `IN`)
- Aggregations with `GROUP BY` and `HAVING`
- ORDER BY

Objective:  
Understand how to combine datasets and perform multi-step analysis across tables.

---

## Key Learnings

- How to extract and filter relevant data efficiently  
- How to summarise large datasets into meaningful insights  
- How to apply conditional logic to enhance analysis  
- How to combine multiple tables to answer complex business questions  
- How to structure queries for clarity and performance  

---

## Outcome

Through these exercises, I developed a strong foundation in SQL, enabling me to:
- Query and manipulate data effectively  
- Perform exploratory data analysis  
- Generate insights to support decision-making  
- Build a base for more advanced data analytics techniques
